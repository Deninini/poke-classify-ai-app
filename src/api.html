API Documentation

                
                

                    
Endpoints

                    

                        

                            Endpoint
                            Method
                            Description
                        

                        

                            /api/identify
                            POST
                            Identify a Pokémon from an image URL
                        

                    

                

                
                

                    
Request Format

                    

                        
{
  "imageUrl": "https://example.com/pokemon.jpg"
}

                    

                

                
                

                    
Response Format

                    

                        
{
  "success": true,
  "pokemon": {
    "id": 25,
    "name": "Pikachu",
    "type": ["Electric"],
    "confidence": 0.98
  }
}

                    

                

                
                

                    
Error Handling

                    

                        
{
  "success": false,
  "error": "Invalid image URL",
  "code": "INVALID_URL"
}
